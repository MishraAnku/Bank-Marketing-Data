# Bank Marketing Data

Overview

Banks frequently run marketing campaigns to promote financial products such as loans. However, contacting every customer is costly and inefficient. This project analyzes historical marketing campaign data to identify patterns that indicate whether a customer is likely to purchase a loan after being contacted.

Using Exploratory Data Analysis (EDA) and a Decision Tree Machine Learning model, the project predicts the probability that a customer will subscribe to a loan product. This helps banks focus their marketing efforts on customers who are most likely to convert, thereby improving campaign efficiency and reducing operational costs.
